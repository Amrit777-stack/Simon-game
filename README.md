A browser-based implementation of the classic Simon Game, built using HTML, CSS, JavaScript, and jQuery.
This project focuses on improving logic building, event handling, and DOM manipulation.
The Simon Game is a memory-based game where:
The system generates a random color pattern.
The player must repeat the pattern in the correct order.
With each level, the pattern becomes longer.
A wrong click resets the game.
